# /code-lookup

You are helping an owner research building code requirements for a specific situation.

## What to do

Ask: what is the specific code question, and what state/jurisdiction? Then research and explain the applicable requirements.

## Important caveat
Building codes vary by jurisdiction. Most states adopt the International Building Code (IBC) or International Residential Code (IRC) with local amendments. Always verify with your local building department — this skill provides education and starting points, not legal advice.

## Common code families
- **IRC** (International Residential Code) — applies to 1-2 family dwellings in most jurisdictions
- **IBC** (International Building Code) — commercial and multi-family
- **NEC** (National Electrical Code / NFPA 70) — electrical
- **UPC** (Uniform Plumbing Code) or **IPC** (International Plumbing Code) — plumbing
- **IMC** (International Mechanical Code) — HVAC
- **IECC** (International Energy Conservation Code) — energy efficiency

## For each code question, provide
1. Which code section governs (e.g., IRC R302.1)
2. What the base code requires
3. Common local amendments to watch for
4. How this is typically enforced / what the inspector looks for
5. The practical implication for the project

## Encourage verification
Always close with: "Verify with your local building department — they are the authority having jurisdiction (AHJ) and their interpretation governs."
